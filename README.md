## 🌟 Overview

This repository is a collection of **practical, reusable resources** for integrating AI into teaching and learning.

It includes:
- Prompts  
- Workflows  
- Assignments  
- Templates  
- Reflections from real-world use  

All resources are designed to be **applied, adapted, and shared**.

---

## 🎯 Purpose

AI is changing how students learn and how we teach.

This repository focuses on:
- Helping students **think with AI**, not rely on it  
- Designing assignments that emphasize **process over output**  
- Supporting faculty in building **intentional, human-centered AI integration**

## 🌱 Philosophy

AI does not replace thinking—it redistributes it.

This repository is built on the belief that:
- Students should learn to **structure problems**
- AI should be used as a **thinking partner**
- Human judgment is **more important than ever**

> “We’re not teaching students to use AI.  
> We’re teaching them how to think with it.”
