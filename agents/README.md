# 🤖 AI Agents for Teaching & Learning

This folder contains a collection of AI-powered agents designed to support student success, improve learning experiences, and enhance instructional workflows in higher education.

Each agent is built with a human-centered approach, focusing on clarity, structure, and actionable outcomes.

---

## 🎯 Purpose

These agents are designed to:

- Support student decision-making and self-regulation  
- Make invisible processes (like time management) visible  
- Provide structured guidance without overwhelming users  
- Integrate into real classroom and LMS environments  

---

## 🧠 Design Philosophy

All agents in this collection follow a consistent set of principles:

### Human-Centered
AI supports thinking—it does not replace it.

### Structured Conversations
Agents guide users step-by-step to reduce cognitive overload.

### Transparent Logic
Calculations, estimates, and outputs are clearly explained.

### Action-Oriented
Every interaction ends with a practical next step.

### Supportive Tone
Agents are encouraging, non-judgmental, and student-focused.

---

## 📂 Available Agents

### 🦅 Eagle Flight Plan
A student-facing agent that helps learners evaluate whether their course schedule realistically fits their available time.

**Key Features:**
- Estimates weekly coursework time  
- Calculates available time based on real-life commitments  
- Provides:
  - Flight Status (Safe, Caution, Danger)  
  - Eagle Score (schedule balance indicator)  
  - Time Reality Chart  
  - Weekly Flight Plan  
  - Reflection prompt
