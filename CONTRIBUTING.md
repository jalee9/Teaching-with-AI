# 🤝 Contributing to Teaching with AI  
### How to share workflows, prompts, and ideas

---

## 🌟 Welcome

Thank you for your interest in contributing!

This repository is designed to be a **shared space for practical AI use in teaching and learning**. Contributions can include:

- Workflows  
- Prompts  
- Assignments  
- Templates  
- Classroom ideas or adaptations  

You do not need to be an expert in GitHub to contribute.

---

## 🎯 What Makes a Great Contribution?

We are looking for resources that are:

- Practical and usable  
- Clearly explained  
- Focused on teaching and learning  
- Aligned with human-in-the-loop AI use  

👉 If you’ve used it in your work or classroom, it’s worth sharing.

---

## 🧠 Before You Start

Ask yourself:

- What problem does this solve?  
- How would someone else use this?  
- What makes this helpful or unique?  

---

## 🛠️ How to Contribute (Beginner-Friendly)

### Option 1: Easiest Way (No GitHub Experience Needed)

1. Send your contribution via:
   - Email  
   - Teams message  
   - Shared document  

2. Include:
   - Title  
   - Description  
   - Any prompts or steps  
   - Optional: screenshots or examples  

👉 I will format and add it to the repository for you.

---

### Option 2: Direct GitHub Contribution

If you are comfortable using GitHub:

#### Step 1: Fork the Repository

- Click the **“Fork”** button (top right of the repo page)

#### Step 2: Add Your Content

- Navigate to the appropriate folder:
  - `/workflows/`
  - `/prompts/`
  - `/assignments/`
  - `/templates/`

- Create a new folder if needed

- Add a `README.md` file using the provided template

#### Step 3: Submit a Pull Request

- Click **“Contribute” → “Open Pull Request”**
- Add a short description of your contribution
- Submit

---

## 🧩 Contribution Guidelines

To keep the repository consistent:

- Use clear titles  
- Follow existing formatting where possible  
- Include examples (when helpful)  
- Keep language professional and accessible  

---

## 🔁 Suggested Format

For workflows and prompts, use the templates provided in the repository:

- `/workflows/_template.md`  
- `/prompts/` (structure coming soon)

---

## 👀 Review Process

All contributions will be:

- Reviewed for clarity and alignment  
- Formatted if needed  
- Added to the appropriate section  

You may be contacted with small suggestions or edits.

---

## 🌱 Philosophy

This repository is built on the belief that:

- AI should support thinking—not replace it  
- Teaching practices should be shared and adapted  
- Learning improves through collaboration  

> “We’re not teaching students to use AI.  
> We’re teaching them how to think with it.”

---

## 🙌 Thank You

Your contribution helps build a shared understanding of how AI can be used responsibly and effectively in education.

Even small ideas can make a big impact.
